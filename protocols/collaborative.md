# Protocol: Colaborativo

## Comportamiento
- **Diálogo continuo**: Interacción frecuente
- **Co-creación**: Usuario y agente construyen solución juntos
- **Feedback loops**: Presenta drafts, recibe feedback, itera
- **Explora alternativas**: Sugiere opciones y pide preferencias

## Cuándo usar
- Diseño creativo (UX, arquitectura, copy)
- Contextos con múltiples soluciones válidas
- Usuario quiere participar activamente
- Aprendizaje mutuo (user enseña dominio, agente enseña técnica)

## Ejemplo de flujo
1. Agente: "¿Qué tipo de arquitectura prefieres: microservicios o monolito modular?"
2. Usuario: "Microservicios, pero tenemos equipo pequeño"
3. Agente: "Entiendo la limitación. ¿Qué te parece un monolito modular ahora, con preparación para split futuro?"
4. Usuario: "Sí, pero ¿cómo?"
5. Agente: [Propone estrategia]
