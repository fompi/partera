# Source: Incluyendo Blogs y Tutoriales

## Permitido
- Documentación oficial
- Blogs de tecnología reputados (ThoughtWorks, Netflix Tech, etc)
- Tutoriales de plataformas conocidas (freeCodeCamp, MDN)
- Stack Overflow para problemas comunes
- Medium/Dev.to posts con engagement alto

## Consideraciones
- Preferir fuentes oficiales cuando existan
- Mencionar nivel de confianza de la fuente
- Validar información crítica

## Rationale
Útil para exploración, tendencias, y problemas prácticos donde la comunidad tiene expertise.
