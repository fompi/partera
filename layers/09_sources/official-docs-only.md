# Source: Solo Documentación Oficial

## Restricción
Solo utilizar información de:
- Documentación oficial del proyecto/tecnología
- Especificaciones (RFCs, estándares ISO, W3C)
- Páginas oficiales de vendors/proveedores
- Papers académicos peer-reviewed

## Prohibido
- Blogs personales o corporativos
- Stack Overflow o foros
- Tutoriales de terceros no oficiales
- Información no verificada

## Rationale
Maximiza precisión y confiabilidad, especialmente en contextos críticos (security, compliance, producción).
